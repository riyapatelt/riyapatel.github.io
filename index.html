02 Mar–Apr 2026 Role: Sole Investigator

36-Day Distributed Credential Attack — M365 Shared Mailboxes

Investigated a 36-day distributed credential attack against 9 Microsoft 365 mailboxes using legacy SMTP authentication. Correctly classified as distributed brute-force / credential stuffing — not password spray — based on target-repetition pattern. Identified the canonical BAV2ROPC user-agent fingerprint and an Azure-hosted attacker IP abusing Microsoft's own infrastructure to evade reputation-based blocks.

77failed sign-ins
58attacker IPs · 23 countries
0confirmed compromises
3MFA-required = password correct

Used a three-query compromise-assessment methodology: tenant-wide legacy-auth SUCCESS check (0 hits), attacker-IP cross-check across all sign-ins (0 successes), and legitimate SUCCESS attribution (all traced to corporate SSE / VPN egress). Authored a phased remediation plan spanning immediate hardening (SMTP AUTH disablement, credential rotation, CA blocks), short-term policy (tenant-wide legacy-auth block, smart-lockout tuning, risk-based CA), and longer-term posture (FIDO2, PIM, phishing-resistant MFA). Also identified a NonInteractiveUserSignInLogs ingestion gap in Graylog as a forward-looking finding.

03 Q1 2026 Role: Lead Investigator

Telerik RAU Handler Threat Hunt — Multi-Server

Graylog-driven threat hunt across 7 web servers following anomalous activity. Mapped behaviour to CVE-2019-18935 (.NET deserialization RCE, CVSS 9.8) and CVE-2017-11317 (arbitrary file upload) via the Telerik RAU handler. Distinguished automated scripted activity from human activity via request-rate and timing heuristics; cross-correlated the same attacker IP appearing under different user contexts across servers.

Behavioural signals that surfaced the hunt: automated session polling on 25-minute intervals, 24 RAU POSTs/second peak from a single account (scripted uploads), first-seen IPs geolocating to cloud infrastructure, and an unauthenticated RAU POST outside any authenticated session. Coordinated with Red Team and developers for remediation.

04 Q2 2025 Role: Investigator · Blue Team

EDR Tampering & SYSTEM Impersonation — Cortex XDR

Investigated an advanced EDR tampering attempt where the Cortex XDR agent was disabled via cyserver.exe /serviceremoveppl without the supervisor password — a PPL-protected-service bypass via SYSTEM token impersonation. Correlated Cortex timeline with Windows Security Logs to reconstruct the attack chain: WMI remote execution → rundll32 loading an MSI-dropped DLL → unsigned msiexec payload → EDR tamper.

Attributed privilege abuse to SeDebugPrivilege, SeLoadDriverPrivilege and SeImpersonatePrivilege — the exact combination needed to tamper with a PPL-protected agent. Flagged Event ID 5379 (Credential Manager reads) minutes before the tamper as classic pre-exfiltration credential staging.

05 Feb 2026 Role: Lead Investigator

Impossible-Travel False Positive → Upstream Geolocation Bug

Triaged an impossible-travel / unusual-location alert (Azure CLI sign-in flagged from Dubai, MFA-passed but CA-invalidated via error code 70044). Investigation traced the flagged IP to the organization's SSE provider egress pool, confirmed via network traffic logs and direct user outreach. Root-caused to an upstream geolocation data accuracy issue.

Produced clean false-positive disposition and — more importantly — engineered a new detection rule to monitor any future non-India/US sign-ins as standing content. The triage → root cause → new detection loop in action.
